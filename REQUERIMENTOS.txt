
# Requerimientos
## Servidores
Servidor de Aplicación: Se requiere un servidor de aplicaciones compatible con Java para ejecutar la aplicación. Se recomienda Apache Tomcat versión 9.0 o superior.
Servidor Web: Se necesita un servidor web para servir los archivos estáticos y manejar las solicitudes HTTP. Se recomienda Apache HTTP Server o Nginx.
## Bases de Datos
Motor de Base de Datos: Se necesita un motor de base de datos relacional compatible con Java. Se recomienda PostgreSQL versión 10 o superior.
## Paquetes Adicionales
Spring Framework: Se utiliza Spring Framework para el desarrollo de la lógica de negocio. Se recomienda utilizar Spring Boot para simplificar la configuración y el desarrollo.
Hibernate: Se utiliza Hibernate como ORM (Object-Relational Mapping) para el acceso a la base de datos desde la aplicación Java.
React.js: Para el desarrollo de la interfaz de usuario se utiliza React.js, por lo que se requiere Node.js y npm para la instalación de dependencias y construcción del proyecto.
## Versiones de Software
Java: Se requiere Java Development Kit (JDK) versión 11 o superior.
Node.js: Se recomienda Node.js versión 12 o superior para el desarrollo de la interfaz de usuario con React.js.
npm: Se recomienda npm versión 6 o superior para la gestión de dependencias de Node.js.
